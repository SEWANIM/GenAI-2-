## 2. 스토리보드 (3씬 구성, 총 10초)

### 🎬 SCENE 1 — "정지된 오늘" (0~3초)
- **비주얼**: 새벽 어스름, 창원 마산항 바다 위 물안개. 러너 한 명 뒷모습, 아직 멈춰 서 있음.
- **카메라**: 잔잔한 슬로우 줌인
- **감정**: 고요, 망설임
- **자막**: "멈춰있던 오늘"
- **오디오**: 잔잔한 앰비언트 + 파도 소리

### 🎬 SCENE 2 — "건너다" (3~7초)
- **비주얼**: 러너가 저도 콰이강의다리(붉은 다리)를 힘차게 달려 건넘. 발밑에서 빛 번짐.
- **카메라**: 측면 트래킹 → 다이나믹
- **감정**: 결심, 폭발적 에너지
- **자막**: "내일로 건너다"
- **오디오**: 비트 상승, 발소리 리듬

### 🎬 SCENE 3 — "새로운 아침" (7~10초)
- **비주얼**: 다리 끝, 창원 도시 위로 일출. 러너 실루엣이 골든아워 빛을 받으며 팔 벌림.
- **카메라**: 로우앵글 → 하늘로 틸트업
- **감정**: 해방, 성취
- **자막**: "NAERIRO — Cross to Tomorrow"
- **오디오**: 음악 절정 → 여운

## 3. AI 생성 파이프라인 (기획 → 프롬프트 → 결과 → 편집)

### STEP 1. 이미지 확정 (크레딧 절약 핵심!)
- 각 씬을 **정지 이미지**로 먼저 생성 → 구도/톤 확정
- 스타일 레퍼런스(sref) 고정으로 3씬 일관성 유지

### STEP 2. 이미지 → 영상 변환
- 확정 이미지를 Image-to-Video로 짧은 모션 부여 (줌/패닝/러닝 모션)

### STEP 3. 오디오 생성
- 배경음악(BGM) + 앰비언트 사운드 AI 생성

### STEP 4. 통합 편집 (CapCut/Premiere)
- 컷 편집, 자막, 색보정, 오디오 레벨 조정만

## 5. 일관성 유지 전략 (cref/sref)

- **화풍**: 시네마틱 새벽 톤, 필름 그레인, 아나몰픽 렌즈 느낌 → sref 고정
- **비율**: 기본 16:9로 생성, 보너스용 9:16 / 1:1 별도 렌더
- **캐릭터**: 러너 뒷모습/실루엣 위주 → 얼굴 일관성 부담 줄이고 cref 보조
- **컬러**: 남보라→골드 그라데이션을 프롬프트에 반복 명시

---

## 6. 크레딧 부족 대응 플랜

- ✅ 6씬 → **3씬**으로 이미 압축 (메시지 밀도 ↑)
- ✅ 정지 이미지 + 짧은 모션(줌/패닝) 중심 구성
- ✅ 동일 sref 고정으로 재생성 횟수 최소화
