# GenAI-2-GenAI 
## 🔧 제작 파이프라인 & 설계 근거

### 전체 순서: 기획 → 생성 → 검수 → 편집/통합

**① 기획 단계**
- NAERIRO 아이덴티티 정의 (타깃: 20대 여행객 / USP: "새벽 기차로 떠나는 창원의 하루")
- 3씬 스토리보드 확정 (S1 새벽역 → S2 달리는 기차 → S3 창원 일출)
- 씬별 프롬프트·내레이션·출력요약 설계

**② 생성 단계** (도구별 역할 분리)
- Midjourney → 각 씬 키프레임 이미지 3종 생성
- Runway(I2V) → 키프레임에 모션 부여
- Suno → 브랜드 톤(잔잔→고조) BGM 생성

**③ 검수 단계**
- 체크리스트: 비율(16:9)/색감(블루→오렌지)/톤 일관성
- 불일치 발견 시 → 수정 로그 기록 후 재생성

**④ 편집/통합 단계**
- 색보정으로 씬 간 톤 통일 → 씬 연결 → 오디오 싱크 → 렌더링

## 🧠 핵심 기술 원리 적용

### 1. Text-to-Image vs Image-to-Video 차이와 선택 기준
- **T2I (Midjourney)**: 텍스트→정지 이미지. 구도·색감·스타일을 **정밀 제어** 가능
- **I2V (Runway)**: 정지 이미지→움직임 생성. 확정된 화면에 **일관된 모션** 부여
- **본 프로젝트 선택 근거**:
  → T2I로 창원 새벽 감성의 키프레임을 **먼저 확정**(스타일 통제)
  → 그 이미지를 I2V에 넣어 **기차 움직임·일출 변화** 추가
  → 만약 T2V(텍스트→영상)만 썼다면 씬마다 스타일이 흔들려 **일관성 확보 어려움**

### 2. 도구 선택 우선순위: 품질 > 제어 > 비용 > 속도
- 브랜드 광고라 **품질**이 1순위 → 이미지 퀄리티 최상급인 Midjourney
- **제어** 2순위 → 프롬프트+`--sref`로 구도/스타일 고정
- **비용** 3순위 → 재생성 횟수 제한으로 관리
- **속도**는 후순위 → 대기열 감수 (품질 위해 트레이드오프)

### 3. 프롬프트 A→B 변화와 기술적 원인
**S3 일출씬 예시**
- A: `Changwon sunrise, train, wide shot`
- B: `Changwon sunrise, train, wide shot, golden hour lighting, warm orange tone, cinematic`
- **결과 변화**: 밋밋했던 화면 → 따뜻한 역광·대비 강조된 시네마틱 화면
- **원인**: 'golden hour', 'warm orange' 토큰이 모델의 **광량·색온도 가중치**를
  높여 조명 연산이 재계산됨 → 분위기 극대화

### 4. 캐릭터/스타일 일관성 제어 방법
- **레퍼런스 고정**: `--sref [시드값]`으로 3씬 모두 동일 스타일 시드 적용
- **파라미터 고정**: `--ar 16:9 --style raw` 전 씬 통일
- **후처리 보정**: 편집 단계에서 LUT 색보정으로 톤 미세 통일

### Q1. 도구 결과물 통합 시 불일치 보정 기준·절차
서로 다른 도구(MJ·Runway·Suno)의 출력은 해상도·색감·톤이 달라짐.
**보정 절차**:
1. **해상도**: 모든 영상을 1920×1080으로 리사이즈 통일
2. **비율**: 16:9 크롭 기준 고정 (전 씬 동일)
3. **색감**: S1(블루)~S3(오렌지) 그라데이션이 자연스럽도록 **LUT 적용**
4. **톤**: BGM 볼륨을 씬 전개에 맞춰 잔잔→고조로 조정
→ 기준은 "**시간 흐름(새벽→일출)의 자연스러운 연결**"

### Q2. 같은 스토리보드를 다른 도구로 만들면?
- **품질**: Sora로 만들면 모션 자연스러움↑, 단 세밀한 구도 **제어력↓**
- **제어**: 현 Runway 조합이 키프레임 기반이라 구도 통제 유리
- **속도**: Pika는 빠르지만 창원 일출의 디테일 표현 부족
- **비용**: Pika/무료툴이 저렴하나 광고 퀄리티엔 미달
→ 결론: 광고 목적엔 **품질·제어 우수한 MJ+Runway 조합이 최적**

### Q3. 크레딧이 부족해진다면?
- **씬 구성**: 3씬 → 2씬(새벽역→일출)으로 축소, 중간 달림씬 삭제
- **도구 선택**: 유료 Runway → 무료 Pika/정지이미지 줌인으로 대체
- **편집 전략**: 영상 생성 최소화, MJ 정지 이미지에 **Ken Burns(줌·팬) 효과**로
  움직임 연출 → 크레딧 절약하며 완성도 유지

### Q4. 60초를 15초로 줄인다면?
- **유지**: S3 창원 일출 클라이맥스 (USP "새벽 기차로 떠나는 창원의 하루" 전달 핵심)
- **삭제/축소**: S2 달리는 기차 전개씬 → 1~2초로 압축
- **메시지 재구성**: 스토리 나열 대신 **일출 임팩트 + 슬로건 + 브랜드명**만 남겨
  15초 안에 브랜드 각인 집중
